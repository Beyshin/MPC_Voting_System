# Frontend - projekt glosowanie

## Wymagania

- Node.js 18+
- npm 9+

## Instalacja

Z poziomu katalogu `frontend`:

```bash
npm install
```

## Uruchomienie lokalne (dev)

```bash
npm run dev
```

Domyslnie aplikacja uruchomi sie pod adresem:

- http://localhost:5173

## Struktura frontendu

```text
frontend/
  public/
    favicon.svg
  src/
    App.jsx
    index.css
    main.jsx
  index.html
  tailwind.config.js
  postcss.config.js
  vite.config.js
  package.json
```
